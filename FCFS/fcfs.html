FIRST COME FIRST SERVE
F.C.F.S.
Basic Theory :

FCFS stands for 'first-come-first-serve'. As the name suggests, the request which comes first will be processed first and so on. The requests coming to the disk are arranged in a proper sequence as they arrive. Since every request is processed in this algorithm, so there is no chance of 'starvation'.
  • Initialise a queue to hold the disk requests.
  • Using enhanced bubble sort technique, sort the all given processes in ascending order according to arrival time in a ready queue.
  • Add all the disk requests to the queue in the order in which they arrive.
  • For each request in the queue, calculate the distance between the current head position and the position of the request.
  • Move the disk head to the position of the request and process the request.
  • Calculate the total head movement by adding up the distance between each request.
  • Return the total head movement.
